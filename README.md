# Healthcare Data Cleaning using Python

## Project Overview
This project demonstrates how Python can be used to clean and prepare healthcare patient data for analysis.

Healthcare datasets often contain missing values, duplicates, and inconsistent formatting. This project shows basic data cleaning techniques used in real-world healthcare and clinical research environments.

## Skills Demonstrated
- Data Cleaning using Pandas
- Handling Missing Values
- Removing Duplicates
- Basic Data Exploration
- Healthcare Data Handling

## Dataset Description
Sample dataset includes:
- Patient ID
- Age
- Gender
- Diagnosis
- Visit Cost
- Missing values simulation

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Why this project?
Clean data is critical before performing analysis in healthcare and clinical research. This project shows how Python can help automate manual Excel cleaning tasks.
